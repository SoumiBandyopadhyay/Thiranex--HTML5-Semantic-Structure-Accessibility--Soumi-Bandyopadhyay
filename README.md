# HTML5 Semantic Portfolio Website

A modern, fully accessible personal portfolio website built using **Semantic HTML5** and **WCAG accessibility guidelines**. This project demonstrates best practices for semantic markup, SEO optimization, and keyboard accessibility.

---

## 📌 Features

* Semantic HTML5 structure
* Responsive layout
* WCAG-compliant accessibility
* SEO-friendly meta tags
* ARIA labels and landmark roles
* Skip-to-content navigation
* Keyboard-accessible navigation
* Accessible contact form
* Proper heading hierarchy
* Lighthouse-friendly structure

---

## 🏗️ HTML5 Semantic Elements Used

* `<header>`
* `<nav>`
* `<main>`
* `<section>`
* `<article>`
* `<footer>`

These elements improve both accessibility and search engine optimization by providing meaningful document structure.

## 🚀 Getting Started

1. Clone the repository.

```bash
https://github.com/SoumiBandyopadhyay/Thiranex--HTML5-Semantic-Structure-Accessibility--Soumi-Bandyopadhyay/tree/main
```

2. Navigate to the project folder.

```bash
cd html5-semantic-portfolio
```

3. Open `index.html` in your preferred web browser.

No additional dependencies or installation steps are required.
